# tba for Go
A Go library for interfacing with The Blue Alliance's API. (Supports APIv3.)

To install `tba`:

    go get github.com/CarlColglazier/tba

Then, to use it in your project:

```go
import (
    "github.com/CarlColglazier/tba"
)
```
